HTML

html
HTML stands for Hyper Text Markup Language HTML is the standard markup language for creating Web pages
  1. HTML describes the structure of a Web page
  2. HTML consists of a series of elements
  3. HTML elements tell the browser how to display the content

paragraph tag

paragraph
The HTML p tag element defines a paragraph. A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

Formatting tags

formatting tags
It is used to style or structure the particular text or content of website
  • bold tag:defines bold text, without any extra importance.
  • Italic tag:The content inside is typically displayed in italic.
  • strong tag:defines text with strong importance
  • emphasied tag:defines emphasized text
  • delete tag:defines text that has been deleted from a document.
  • mark tag:defines text that should be marked or highlighted:
  • sub tag:Subscript text can be used for chemical formulas, like H2O

Block and Inline elements

Block Elements
A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
Inline Elements
An inline element does not start on a new line. An inline element only takes up as much width as necessary.

Tables

tables
An HTML Table is an arrangement of data in rows and columns in tabular format.
  1. table tag:Defines the structure for organizing data in rows and columns within a web page.
  2. tr tag:Represents a row within an HTML table, containing individual cells
  3. th tag:Shows a table header cell that typically holds titles or headings.
  4. td tag:Represents a standard data cell, holding content or data.

Image tags

Image
  • Images can improve the design and the appearance of a web page.
  • The HTML tag is used to embed an image in a web page.
  • Images are not technically inserted into a web page; images are linked to web pages. The tag creates a holding space for the referenced image.
  • The tag is empty, it contains attributes only, and does not have a closing tag.
  • The tag has two required attributes: src - Specifies the path to the image alt - Specifies an alternate text for the image

Anchor tag

anchor tag
The HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

Structure of URL

structure of url
A URL consists of ten parts: the scheme, subdomain, top-level domain, second-level domain, subdirectory, parameter, port, path, query, and fragment.

Description List
A description list is a list of terms, with a description of each term. The dl tag defines the description list, the dt tag defines the term (name), and the dd tag describes each term
